# Guide de Timing et Rythme Vidéo

## Règles fondamentales

### Calcul mots/durée (voix-off)

| Rythme | Mots/seconde | Mots/minute | Usage |
|--------|--------------|-------------|-------|
| Lent | 2.0 | 120 | Émotionnel, dramatique |
| Normal | 2.5 | 150 | Explicatif, standard |
| Rapide | 3.0 | 180 | Énergique, pub dynamique |
| Très rapide | 3.5+ | 210+ | Shorts, cuts rapides |

### Tableau de conversion rapide

| Durée vidéo | Mots (normal) | Mots (rapide) |
|-------------|---------------|---------------|
| 15 sec | 35-40 mots | 45-50 mots |
| 30 sec | 70-80 mots | 90-100 mots |
| 45 sec | 105-120 mots | 135-150 mots |
| 60 sec | 140-160 mots | 180-200 mots |
| 90 sec | 210-240 mots | 270-300 mots |

---

## Rythme des cuts (montage)

### Vidéos dynamiques (Shorts, pubs)

| Type de cut | Fréquence | Effet |
|-------------|-----------|-------|
| Cut rapide | Toutes les 2-3 sec | Énergie, attention maintenue |
| Cut moyen | Toutes les 4-6 sec | Équilibré, professionnel |
| Cut lent | Toutes les 8-12 sec | Dramatique, émotionnel |

### Recommandations par type

| Type de vidéo | Fréquence cuts | Style |
|---------------|----------------|-------|
| Short/Reel | 2-3 sec | Ultra-dynamique |
| Pub marketing | 3-4 sec | Dynamique |
| Témoignage | 4-6 sec | Authentique |
| Tutoriel | 5-8 sec | Clair, digestible |
| Story | 6-10 sec | Immersif |

## Structure temporelle par durée

### Format 30 secondes
```
[0:00-0:03] Hook (10%)
[0:03-0:08] Problème/Contexte (17%)
[0:08-0:22] Corps/Solution (47%)
[0:22-0:27] Preuve/Résultat (17%)
[0:27-0:30] CTA (10%)
```

## Règles du hook (0-3 secondes)

- **Seconde 1** : Pattern interrupt visuel OU audio
- **Seconde 2** : Promesse ou question
- **Seconde 3** : Transition vers le contenu

**Formats visuels efficaces pour le hook :**
- Texte gros + voix forte
- Zoom rapide sur visage
- Mouvement inattendu
- Couleur vive / contraste fort

## Pauses et respiration

| Type | Durée | Quand l'utiliser |
|------|-------|------------------|
| Micro-pause | 0.3-0.5 sec | Entre les phrases |
| Pause courte | 0.5-1 sec | Avant un point important |
| Pause dramatique | 1-2 sec | Avant une révélation |

---

## Checklist timing avant export

- [ ] Le hook est dans les 3 premières secondes
- [ ] Aucun "temps mort" de plus de 2 secondes
- [ ] Le CTA commence avant les 5 dernières secondes
- [ ] Le nombre de mots correspond à la durée cible
- [ ] Les cuts suivent le rythme recommandé
