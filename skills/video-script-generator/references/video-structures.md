# Structures Narratives Vidéo

## Vue d'ensemble

Chaque type de vidéo a une structure optimale. Choisir la bonne structure selon l'objectif.

---

## 1. Structure AIDA (Pub marketing)

**Usage** : Publicités, vidéos de vente, promotions
**Durée idéale** : 30-60 secondes

| Phase | Durée | Contenu |
|-------|-------|---------|
| **A**ttention | 3 sec | Hook percutant, pattern interrupt |
| **I**ntérêt | 10-15 sec | Problème identifié, empathie |
| **D**ésir | 15-25 sec | Solution + bénéfices + preuve |
| **A**ction | 5-10 sec | CTA clair et urgent |

**Exemple structure 45 sec** :
```
[0:00-0:03] ATTENTION : "Tu perds 5h/semaine à faire ça manuellement."
[0:03-0:15] INTÉRÊT : Description du problème, conséquences
[0:15-0:35] DÉSIR : Présentation solution, résultats clients
[0:35-0:45] ACTION : "Teste gratuitement → lien en bio"
```

---

## 2. Structure PAS (Problem-Agitate-Solve)

**Usage** : Vidéos explicatives, contenu éducatif orienté solution
**Durée idéale** : 45-90 secondes

| Phase | Durée | Contenu |
|-------|-------|---------|
| **P**roblème | 10-15 sec | Identifier le problème clairement |
| **A**gitation | 15-25 sec | Amplifier la douleur, conséquences |
| **S**olution | 20-40 sec | Présenter la solution + comment |

**Exemple structure 60 sec** :
```
[0:00-0:15] PROBLÈME : "La prospection manuelle, c'est l'enfer."
[0:15-0:35] AGITATION : Temps perdu, opportunités ratées, stress
[0:35-0:55] SOLUTION : Automatisation + démo rapide
[0:55-1:00] CTA : Appel à l'action
```

---

## 3. Structure Témoignage (BAB)

**Usage** : Témoignages clients, études de cas
**Durée idéale** : 30-60 secondes

| Phase | Durée | Contenu |
|-------|-------|---------|
| **B**efore | 10-15 sec | Situation avant (problème vécu) |
| **A**fter | 15-25 sec | Situation après (résultats obtenus) |
| **B**ridge | 10-15 sec | Comment (le produit/service) |

**Exemple structure 45 sec** :
```
[0:00-0:12] BEFORE : "Avant, je passais 3h/jour sur mes factures."
[0:12-0:30] AFTER : "Maintenant, c'est automatique. Je gagne 15h/semaine."
[0:30-0:45] BRIDGE : "Grâce à [Produit], tout est simplifié."
```

---

## 4. Structure Tutorial/How-To

**Usage** : Tutoriels, guides pratiques, tips
**Durée idéale** : 45-90 secondes

| Phase | Durée | Contenu |
|-------|-------|---------|
| Hook + Promesse | 3-5 sec | Ce qu'ils vont apprendre |
| Contexte rapide | 5-10 sec | Pourquoi c'est important |
| Étapes | 25-60 sec | 3-5 étapes claires |
| Récap + CTA | 5-10 sec | Résumé et appel à l'action |

**Exemple structure 60 sec** :
```
[0:00-0:05] HOOK : "Automatise ta prospection en 3 étapes."
[0:05-0:12] CONTEXTE : Pourquoi automatiser
[0:12-0:50] ÉTAPES : Étape 1, 2, 3 avec visuels
[0:50-1:00] RÉCAP + CTA : "Maintenant, à toi de jouer. Lien en bio."
```

---

## 5. Structure Story (Narrative)

**Usage** : Personal branding, storytelling, connexion émotionnelle
**Durée idéale** : 60-90 secondes

| Phase | Durée | Contenu |
|-------|-------|---------|
| Setup | 10-15 sec | Contexte, personnage, situation initiale |
| Conflit | 20-30 sec | Problème, obstacle, tension |
| Résolution | 20-30 sec | Comment le problème a été résolu |
| Leçon | 10-15 sec | Morale, apprentissage, CTA |

---

## 6. Structure Liste/Ranking

**Usage** : Top X, comparatifs, listes de ressources
**Durée idéale** : 30-60 secondes

| Phase | Durée | Contenu |
|-------|-------|---------|
| Hook + Annonce | 3-5 sec | "Voici X [éléments] pour [objectif]" |
| Items | 20-45 sec | Chaque item avec mini-explication |
| Bonus/CTA | 5-10 sec | Item bonus ou appel à l'action |
