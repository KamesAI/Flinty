# Bibliothèque de Hooks Vidéo

## Principes du hook parfait

- **Durée** : 1-3 secondes maximum
- **Objectif** : Stopper le scroll, créer la curiosité
- **Règle d'or** : Promettre une transformation ou révéler un problème

---

## Hooks par catégorie

### 🔥 Pattern Interrupt (Choc)
```
"Arrête tout."
"Tu fais cette erreur tous les jours."
"J'ai failli tout perdre."
"Ce que personne ne te dit sur [X]..."
"Supprime cette app maintenant."
```

### 🎯 Résultat / Transformation
```
"J'ai fait [X]€ en [Y] jours."
"Voici comment j'ai [résultat] sans [effort]."
"De 0 à [X] en [durée]."
"Cette méthode m'a fait gagner [X] heures/semaine."
"Avant/Après : [transformation]."
```

### ❓ Question provocante
```
"Tu savais que [fait surprenant] ?"
"Pourquoi personne ne parle de ça ?"
"C'est quoi ton excuse ?"
"Tu préfères [option A] ou [option B] ?"
"Et si je te disais que [promesse] ?"
```

### 🚫 Problème / Douleur
```
"Tu perds [X] heures par semaine à faire ça."
"90% des [cible] font cette erreur."
"Si tu fais [action], arrête immédiatement."
"Le vrai problème, c'est pas [X], c'est [Y]."
"Tu te demandes pourquoi [problème] ?"
```

### 🎁 Valeur immédiate
```
"Voici [X] outils gratuits pour [objectif]."
"Le hack que j'aurais aimé connaître avant."
"Copie cette méthode exacte."
"En 30 secondes, tu vas apprendre [X]."
"Le template que j'utilise pour [objectif]."
```

### 📢 Témoignage / Preuve sociale
```
"Mon client a [résultat] en [durée]."
"Ils m'ont dit que c'était impossible."
"Voici ce qui s'est passé quand j'ai [action]."
"[Nombre] personnes utilisent déjà cette méthode."
"Le retour de [client] m'a choqué."
```

### 🤫 Secret / Exclusivité
```
"Ce que les [experts] ne veulent pas que tu saches."
"La stratégie cachée des [top performers]."
"Personne ne parle de cette fonctionnalité."
"Le secret le mieux gardé de [industrie]."
"J'aurais dû garder ça pour moi, mais..."
```

## Hooks spécifiques par type de vidéo

### Pour témoignage SaaS/Produit
```
"Ce logiciel a changé ma façon de travailler."
"Avant [Produit], je perdais [X] heures."
"Mon équipe ne peut plus s'en passer."
"ROI de [X]% en [durée] avec [Produit]."
"J'étais sceptique, jusqu'à ce que..."
```

### Pour pub marketing
```
"[Problème] ? C'est fini."
"Imagine si tu pouvais [bénéfice]."
"Stop. Tu mérites mieux que [situation actuelle]."
"[Nombre] [cible] ont déjà fait le switch."
"La solution existe. La voici."
```

### Pour contenu éducatif/tips
```
"Le tutoriel que j'aurais aimé avoir."
"Apprends ça en [durée]."
"L'erreur #1 des débutants en [domaine]."
"Voici exactement comment faire [action]."
"3 étapes pour [objectif]."
```

## Règles de sélection du hook

1. **Connaître la douleur** → Le hook doit toucher un point sensible
2. **Être spécifique** → Les chiffres et détails augmentent la crédibilité
3. **Créer un gap** → Curiosité entre ce qu'ils savent et ce qu'ils vont apprendre
4. **Tester** → Un hook peut transformer les performances d'une vidéo
