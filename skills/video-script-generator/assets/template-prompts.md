## Conseils d'utilisation

### Pour Midjourney
1. Copier le prompt tel quel
2. Ajouter `/imagine prompt:` avant
3. Ajuster le `--ar` si besoin
4. Utiliser `--v 6` pour la dernière version

### Pour Leonardo AI
1. Coller dans le champ prompt
2. Sélectionner le modèle adapté (Phoenix, Kino XL...)
3. Ajuster les dimensions selon le ratio
4. Activer "Prompt Magic" si disponible

### Pour DALL-E 3
1. Simplifier légèrement le prompt si trop long
2. Spécifier le style clairement
3. Demander des ajustements via conversation
